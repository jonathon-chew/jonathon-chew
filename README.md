## Featured Projects

### [go-pipepeek](https://github.com/jonathon-chew/go-pipepeek)
Pipe inspector and debugger for UNIX pipelines. It passes stdin to stdout unchanged while showing live stats, making it useful for understanding large streams without breaking the pipe.

### [go-repoflow](https://github.com/jonathon-chew/go-repoflow)
A Git and GitHub workflow helper focused on real repository maintenance: TODO tracking, issue linking, repo health checks, version tags, and cross-repo visibility from the terminal.

### [go-codecount](https://github.com/jonathon-chew/go-codecount)
A directory scanner that counts files and lines across multiple languages, with readable terminal output and a practical CLI-first design.

### [Aphrodite](https://github.com/jonathon-chew/Aphrodite)
A small Go package for adding color and formatting to terminal output. Built to be simple enough to drop into other CLI tools without friction.

### [Rat](https://github.com/jonathon-chew/Rat)
A simplified `bat`-style file viewer in Go that pretty-prints files with syntax highlighting and fast terminal-friendly output.
